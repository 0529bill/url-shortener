# URL Shortener (WIP)

### Project tech stack

- Frontend

  1. Typescript
  2. React context-api, useReducer
  3. Vite (replace craco & react-script)
  4. axios
  5. ant design(ui library)

- Backend
  1. node.js + express.js
  2. JWT
  3. google oauth
  4. mongoDB + mongoose

---

### Project feature

1. shortened any URL
2. provides URL's qr-code
3. provides URL's click count
4. google analytics
